Install Heroku command line tool

```
$ sudo snap install heroku --classic
```

then

```
$ heroku create
```

or, in case of existing remote:

```
$ git remote add heroku https://git.heroku.com/guarded-atoll-05288.git
```

```
$ heroku login
```
